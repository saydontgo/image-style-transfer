# 风格迁移实验报告模板

## 1. 题目

基于深度学习的图像风格迁移实验

## 2. 算法理论介绍

本实验采用 Johnson 等人提出的快速风格迁移方法。该方法核心思想如下：

- 使用一个前馈卷积神经网络直接学习“内容图像 -> 风格化图像”的映射。
- 训练时不直接用像素级误差，而是使用预训练 VGG 网络提取高层特征。
- 内容损失用于约束输出图像保持原图的语义结构。
- 风格损失通过 Gram Matrix 统计特征相关性，使输出图像接近目标艺术风格。
- 总变分损失用于提升平滑性，减少噪声与伪影。

可写公式：

```text
L_total = alpha * L_content + beta * L_style + gamma * L_tv
```

其中：

- `alpha` 是内容损失权重
- `beta` 是风格损失权重
- `gamma` 是总变分损失权重

## 3. 代码介绍

本实验代码由以下模块组成：

- `train.py`
  - 负责训练单风格模型
  - 输入为内容数据集与一张风格图
- `stylize.py`
  - 负责使用训练好的模型对图片进行风格迁移
- `compare_models.py`
  - 负责将公开预训练模型与自训练模型在相同内容图上的效果进行可视化对比
- `style_transfer/models/transformer_net.py`
  - 前馈风格迁移网络
- `style_transfer/models/loss_network.py`
  - VGG 感知损失网络与 Gram Matrix 计算

可说明的实现要点：

- 使用 `InstanceNorm2d` 提升风格迁移稳定性
- 使用 `AMP mixed precision` 降低显存占用
- 使用单风格单模型设计，便于完成“两种风格”实验

## 4. 实验设置

### 4.1 硬件环境

- GPU：NVIDIA 5080 16GB
- 系统环境：Windows 11 + WSL2
- 深度学习框架：PyTorch

### 4.2 数据集

- 内容数据集：MS-COCO
- 风格图 1：`mosaic.jpg`
- 风格图 2：`candy.jpg`

### 4.3 训练参数

可填写示例：

```text
image_size = 256
style_size = 512
batch_size = 8
epochs = 2
subset_size = 20000
learning_rate = 2e-4
style_weight = 1e5
content_weight = 1.0
tv_weight = 1e-6
```

### 4.4 对比方案

- 方案 A：下载的公开预训练模型
- 方案 B：自己训练的模型
- 对比原则：
  - 使用同一批测试内容图
  - 尽量使用相同架构
  - 尽量使用相同内容训练集

可以从以下角度分析：

- 原图结构是否被较好保留
- 颜色迁移是否自然

### 5.3 超参数影响分析

可以讨论：

- `style_weight` 增大后，风格更明显，但原图结构可能减弱
- `content_weight` 增大后，内容更清晰，但风格可能偏弱
- `tv_weight` 过大时图像会变平滑甚至发糊
- `subset_size` 增大后结果更稳定，但训练时间更长

## 6. 自己的见解和讨论

可参考以下思路展开：

- 快速风格迁移比 Gatys 优化式方法推理更快，更适合实际应用
- 但快速方法通常需要“一种风格训练一个模型”，灵活性不如优化式方法
- 单卡训练时，合理控制图像分辨率和 batch size 很重要
- 自训练模型如果训练轮数和数据量不足，通常会弱于公开成熟模型
- 若后续继续优化，可以考虑：
  - 增加训练集规模
  - 延长训练轮数
  - 更细致地调节 style/content loss 权重
  - 尝试多风格统一模型

## 7. 结论

可总结：

- 成功实现了基于 PyTorch 的图像风格迁移
- 完成了至少两种艺术风格实验
- 完成了公开模型与自训练模型对比
- 验证了感知损失方法在图像风格迁移任务中的有效性
